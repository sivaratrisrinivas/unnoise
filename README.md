# unnoise

CPU-only diffusion tutorial for watching noise turn into meaning step by step.

## Run the UI

```bash
.venv/bin/python app.py
```

Then open the page in your browser and press **Generate progression**.

## Scripts

- `scripts/generate_single_image.py`
- `scripts/generate_denoising_progression.py`
- `scripts/visualize_denoising_progression.py`
